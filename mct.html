MP3 Analysis Report
MP3 변환은 완전한 오디오→MIDI 변환이 아니라 근사 변환입니다. 단순한 8비트/칩튠 음원에서 가장 잘 작동하며, 저작권이 있는 MP3는 권리가 있을 때만 사용하세요.

.mct는 실제 오디오가 아니라 BPM, 무드, 스케일, 코드 진행, 패턴 데이터를 저장하는 자체 포맷입니다. HTML 게임이나 시뮬레이터에 넣으면 Web Audio가 실시간으로 8비트 음악을 합성합니다.
MP3 → MCT는 연구용/창작 보조용 근사 기능입니다. 원본 음원의 저작권과 이용 권한은 사용자가 직접 확인해야 합니다.
